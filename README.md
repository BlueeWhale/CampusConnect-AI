# 🚀 CampusConnect AI

## 💡 Description
A smart campus management system with AI features.

## 🧩 Modules
- Frontend (HTML/CSS/JS)
- Backend (Node.js API)
- AI Service (Python ML models)

## ⚙️ Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- Python (AI/ML)
- MongoDB (optional)

## 🚀 How to Run

### Frontend
Open `frontend/index.html`

### Backend
```bash
cd backend
npm install
node server.js

AI Service
cd ai-service
pip install -r requirements.txt
python app.py
